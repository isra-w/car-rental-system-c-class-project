## 🎯 About the Project

Welcome to the **Car Rental System** - a comprehensive C++ class project designed to manage car rental operations. This system demonstrates object-oriented programming principles with practical application in real-world scenarios.

### Project Goals
- ✅ Implement robust car rental management functionality
- ✅ Demonstrate C++ class design and architecture
- ✅ Create an interactive user interface
- ✅ Handle business logic for car reservations

## 🚀 Getting Started

### Prerequisites
- **C++17** or later
- **CMake 3.10+** (recommended)
- **g++** or **clang** compiler
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/isra-w/car-rental-system-c-class-project.git
   cd car-rental-system-c-class-project
   ```

2. **Build the project**
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

3. **Run the application**
   ```bash
   ./CarRentalSystem
   ```
